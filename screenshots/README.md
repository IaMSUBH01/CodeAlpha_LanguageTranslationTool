# Screenshots Directory

This directory contains screenshots of the AI Language Translation Tool application.

## How to Add Screenshots

After running `streamlit run app.py`, take screenshots of:

1. `main_ui.png` — The main application interface
2. `translation_demo.png` — A translation in action
3. `tts_demo.png` — The Text-to-Speech audio player

Screenshots can be added with tools like:
- macOS: `Cmd + Shift + 4`
- Windows: `Win + Shift + S`
- Linux: `gnome-screenshot`
